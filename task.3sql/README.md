# Task 3: SQL Basics

## Objective
The objective of this task is to demonstrate basic SQL operations including filtering, sorting, aggregation, and conditional queries using MySQL.

---

## Dataset
- **Dataset:** Superstore Sales Dataset
- **Format:** CSV
- **Imported Using:** MySQL Workbench 8.0
- **Table Name:** `superstore`

The dataset contains sales transaction data such as product category, sales amount, order date, and customer information.

---

## Tools Used
- MySQL Workbench 8.0
- MySQL Database

---

## SQL File
### task3.sql
The `task3.sql` file contains all SQL queries written to complete Task 3.  
Each query is commented to explain its purpose.

Queries included:
- Viewing and verifying data using `SELECT`
- Filtering records using `WHERE`
- Sorting data using `ORDER BY`
- Aggregating data using `GROUP BY`
- Filtering aggregated results using `HAVING`
- Date range filtering using `BETWEEN`
- Pattern matching using `LIKE`

---

## Output File
- **sales_summary.csv**
  - Exported result of category-wise total sales using aggregation.

- Understanding relational data querying
- Practical use of aggregate functions
